迁移-软件安装

由于电脑外借之后被重做了系统，个人目前也有更换30系笔电的打算。为了避免下次迁移时像这次恢复到我之前使用时一样痛苦，将常用的软件安装包备份到 gitee，百度云。至于配置信息则通过这篇博客来进行记录也顺便安利一下一些软件，同时也欢迎各位给我安利好用的软件。

软件

软件名称 简介 官网地址
Microsoft Edge Chromium 内核，同步方便，可装扩展，吃性能较少。 https://www.microsoft.com/zh-cn/edge
scoop Windows包管理器，没代理不推荐。 https://scoop.sh/

vscode

  1. 如果需要设置语言为中文，在扩展中安装chinese包即可。
  2. 可以直接在终端中使用code命令用vscode来打开文件。例如`code test.py

Typora

  1. 将Typora设置为打开markdown(.md)文件的默认打开应用。可以在cmder使用start or .\命令来打开md文件。 start test.md或者.\test.md。ps：.\后无空格，且在cmder中使用tab不会自动补全。

Mathpix Snipping Tool

  2. ocr识别后可自行修改，支持导出到word，或png图片。

问题和建议

  1. 如果在软件的使用中遇到问题，在此之请您先阅读如何提问。
  2. 打开显示文件名后缀，显示隐藏的文件、文件夹和驱动器。
  3. 在桌面保留回收站和当前要处理的文件即可。
    • 将程序固定到开始屏幕和少量任务栏
    • 创建一个文件夹将其添加到环境变量中，将常用软件的快捷方式复制到该文件夹下，修改简单的名称则可用win+r打开运行后，输出缩写名运行。
